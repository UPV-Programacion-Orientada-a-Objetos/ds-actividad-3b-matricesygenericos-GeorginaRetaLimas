### 🎲 Conceptos aprendidos

#### 💣 Destructor
Un destructor es un método especial que se ejecuta **automáticamente** cuando un objeto se destruye. Su sintaxix es:

~NombreClase(){
    // Acciones o cosas que se hagan cuando se destrye
}
